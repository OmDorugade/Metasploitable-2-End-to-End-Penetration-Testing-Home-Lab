# <div align="center">NFS (Port 111 & 2049) Enumeration & Exploitation</div>

<div align="center">
This target exposes RPC services on port 111 (rpcbind) and NFS on port 2049. Misconfigured NFS shares allowed unauthorized access, enabling privilege escalation by injecting SSH keys into the target system.
