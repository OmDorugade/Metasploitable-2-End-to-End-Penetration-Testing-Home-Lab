## 1. SMB Reconnaissance using Nmap

### 🔍 Command Used

nmap -p- -sV -sC <target-ip>

### 📌 Explanation

- `-p-` → Scan all ports  
- `-sV` → Detect service versions  
- `-sC` → Run default scripts  

### 🎯 Purpose

Identify SMB services and detect vulnerable Samba versions.

### 📊 Key Findings

- Port 139 → netbios-ssn (Samba smbd 3.X - 4.X)  
- Port 445 → netbios-ssn (Samba smbd 3.0.20-Debian)  

👉 Indicates a potentially vulnerable Samba version.

## 2. Searching for Exploits

### 🔍 Command Used

searchsploit Samba 3.0.20

### 📌 Explanation

Search for known exploits related to the detected Samba version.

### 📊 Result

- Samba "username map script" Command Execution  
- Remote Heap Overflow  

👉 The username map script exploit is highly reliable.
